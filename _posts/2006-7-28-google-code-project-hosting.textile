Google goes Sourceforge: "Google Code - Project Hosting":http://code.google.com/hosting/. Google gaat opensource projecten hosten. Binnenkort wordt het moeilijk om iets te vinden wat ze *niet* doen. :P

Het zal wel marcheren mijn gedacht, het is van "Google":http://www.google.com.
